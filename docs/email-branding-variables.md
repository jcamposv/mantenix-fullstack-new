## Variables de Branding para MailerSend

- `brand_name`
- `brand_logo_url`
- `brand_logo_small_url`
- `brand_primary_color`
- `brand_secondary_color`
- `brand_background_color`
- `brand_font`

Ejemplos de uso en MailerSend:

- Logo principal: `{{ brand_logo_url }}`
- Nombre de marca en asunto: `{{ brand_name }}`
- Color primario CSS inline: `{{ brand_primary_color }}`
